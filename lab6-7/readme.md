## Алгоритм

1. Сначала нужно создать два списка чисел одинаковой длины — `a` и `b`.

2. Проверить, что в обоих списках одинаковое количество элементов.
   Если количество не совпадает, нужно вывести сообщение об ошибке и остановить выполнение.

3. Создать пустой список `result`, в который будут записываться суммы чисел.

4. Посчитать суммы элементов из двух списков с помощью цикла `while`.
   Пока индекс меньше длины списка, брать по одному числу из каждого списка, складывать их и записывать в новый список.

5. После завершения цикла вывести результат сложения, полученный с помощью `while`.

6. Затем снова пройти по спискам с помощью цикла `for`.
   На каждом шаге брать по одному числу из каждого списка, складывать и записывать сумму в тот же список `result`.

7. После этого вывести результат сложения, полученный с помощью `for`.

8. Когда оба результата выведены, программа завершается.
